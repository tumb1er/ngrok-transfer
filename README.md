# ngrok-transfer
Docker image for [transfer.sh](https://transfer.sh/) + [ngrok](https://ngrok.com/) pair

Usage:
```
docker run --rm -it --env NGROK_AUTH=<ngrok_auth_token> tumb1er/ngrok-transfer
```
